FAQ GalXy

Apa itu GalXy Private Server?

GalXy PS adalah Private Server Growtopia bertema Galaksi & Fantasi yang dirancang untuk memberikan pengalaman bermain yang lebih seru, stabil, dan penuh dengan item kustom unik.

Bagaimana cara bermain di GalXy PS?

Cukup download "All Host Files", masukkan ke folder host perangkatmu (system32/drivers/etc atau di Android menggunakan Virtual Hosts), lalu login Growtopia seperti biasa!

Apakah server ini aman?

Sangat aman. Kami tidak pernah meminta password Anda di luar form login. Pastikan Anda tidak membagikan data akun Anda kepada siapapun di dalam game.
